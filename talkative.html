Talkative :-0

Présentation du projet

Le projet Talkative permet aux bloggeurs et aux fournisseurs de contenu sur le Web d'ajouter un fil de discussion sur leurs articles et leurs pages. Une personne souhaitant commenter un article, le fera directement sur la page concernée via le service de Talkative.

Plus qu'une fonctionnalité d'ajout de commentaires, Talkative permettra de créer des communautés autour de centres d'intérêts (exemple : navigation entre les différents articles intégrant les services Talkative).

Fonctionnalités souhaitées (à revoir)
  • Création d'un compte pour un auteur
  • Ajout d'un fil de discussion sur une page (Web Service)
  • Ajout de commentaires par le service Talkative
  • Gestion d'un fil de discussion par son auteur
  • Possibilité de noter la pertinence des articles
  • Création d'un compte pour les commentateurs
Technologies envisagées
  • Web services RESTful
  • Intégration par JSON et HTML 5
